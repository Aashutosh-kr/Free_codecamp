Introduction

HTML is a markup language used to write websites which we visit or see on Internet. HTML means Hyper Text Markup Language.

Structure

The HTML document has a general Structure like this :-

<html>

<head>

<title>Title of Document</title>

</head>

<body>

<h1>This is main heading.</h1>

<p>This is some paragraph of content of the website.</p>

<h2>This is sub-heading for the main document.</h2>

<p>This is another paragraph of content.</p>

</body>

</html>

In which the <html></html> tags is the container tag which contains all other elements of the document.

The HTML element contains <head> </head> tags, which is not displayed in the browser's window. The HEAD element contains the metadata of the HTML document. It also contains a TITLE element which contains the title of the document, which is displayed on the top of the browser's window or the tab of the browser if the browser supports multitab functionality.

Next is the BODY element which contains all the other elements of HTML document. All the contents which we see in the browser's window is contained within the BODY element.

The elements contained in the BODY element are discussed in the subsequent sections.
