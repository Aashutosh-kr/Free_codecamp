Introduction


HTML is a markup language used to write websites which we visit or see on Internet. HTML means Hyper Text Markup Language.

Structure


The HTML document has a general Structure like this :-

<html>

<head>

<title>Title of Document</title>

</head>

<body>

<h1>This is main heading.</h1>

<p>This is some paragraph of content of the website.</p>

<h2>This is sub-heading for the main document.</h2>

<p>This is another paragraph of content.</p>

</body>

</html>

Title of Document X |


This is main heading.

This is some paragraph of content of the website.

This is sub-heading for the main document.

This is another paragraph of content.

In which the <html> </html> tags is the container tag which contains all other elements of the document.

The <HTML> element contains <head> </head> tags, which is not displayed in the browser's window. The <HEAD> element contains the metadata of the HTML document. It also contains a <TITLE> element which contains the title of the document, which is displayed on the top of the browser's window or the tab of the browser if the browser supports multitab functionality.

Next is the <BODY> element which contains all the other elements of HTML document. All the contents which we see in the browser's window is contained within the BODY element.

The elements contained in the <BODY> element are discussed in the subsequent sections.

Text


We can use different levels of headings, font sizes, font families and also different formatting with the texts.


Headings


We can use six different levels of Headings in HTML. Starting with <h1> to <h6> The subsequent headings are rendered from larger to smaller in browsers by default. The <h1> being the largest and the <h6> smallest.

Paragraphs


We can also write pragraphs by using <p> </p> tags.

By default the different paragraphs are shown in new line by the browsers.

We can format the text differently by using different HTML tags as follows.

Text Formattings

  • Bold - By using <b> </b> tags.
  • Italic - By using <i> </i> tags.
  • Superscript - By using <sup> </sup> tags.
  • Subscript - By using <sub> </sub> tags.
  • Line breaks - by using <br /> tag.
  • Horizontal line - by using <hr /> tag.
  • Strong text - by using <strong> </strong> tags.
  • Emphasis text - by using <em> </em> tags.
  • Block quote of texts - by using <blockquote> <blockquote> tags.
  • Inline quote - by using <q> </q> tags.
  • Abbreviations or Accronym - by using <abbr> </abbr> tags and title attribute for its full fom.

    Hover over HTML in result to see its full form.

  • Insertion of text - by using <ins> </ins> tags. Deletion of text - by using <del> </del> tags.
  • No longer relevence - by using <s> </s> tags.

Lists


We can use Three different types lists in HTML according to our needs.

Each list items in an Ordered or Unordered list must contain <li> </li> tags.


  1. Ordered Lists

    Ordered lists are created with the ol element.

    <p>To </p>

    <ol>

    <li>Lorem, ipsum.</li>

    <li>Lorem, ipsum.</li>

    <li>Lorem, ipsum.</li>

    <li>Lorem, ipsum.</li>

    <li>Lorem, ipsum.</li>

    <ol/>

    To

    1. Lorem, ipsum.
    2. Lorem, ipsum.
    3. Lorem, ipsum.
    4. Lorem, ipsum.
    5. Lorem, ipsum.

  2. Unordered Lists

    Unordered lists are created with the ul element.

    <p>Fruits</p>

    <ul>

    <li>Mango</li>

    <li>Pineapple</li>

    <li>Grapes</li>

    <li>Kiwi</li>

    <li>Oranges</li>

    <ul/>

    Fruits

    • Mango
    • Pineapple
    • Grapes
    • Kiwi
    • Oranges

  3. Definition Lists

    Definition lists are created with the dl element. Inside the <dl> element we use the <dt> </dt> and <dd> </dd> tags to write the Definition term and the Definition of the term respectively.

    <dl>

    <dt>Lorem, ipsum.</dt>

    <dd>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nostrum, eaque!</dd>

    <dt>Lorem, ipsum.</dt>

    <dd>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nostrum, eaque!</dd>

    <dl/>
